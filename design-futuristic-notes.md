# Design Futurista Hub - Notas de Pesquisa

## Referências Analisadas
1. Apple Vision Pro Dashboard - glassmorphism escuro com blur, cards flutuantes, cantos arredondados grandes
2. Glassmorphism Light Dashboard - fundo azul claro gradiente, cards brancos com blur/transparência, sidebar com glassmorphism
3. Sci-fi HUD interfaces - elementos holográficos, linhas de circuito, gradientes neon

## Conceito Escolhido: "Vision Pro meets Engineering"
- Fundo: gradiente mesh suave (azul claro → branco → dourado sutil)
- Cards: glassmorphism com backdrop-blur, bordas semi-transparentes, cantos muito arredondados (24px)
- Animações CSS: hover com scale + glow sutil, entrada com fade-in staggered
- Tipografia: números grandes e bold (estilo dashboard financeiro), labels pequenos
- Cores FC: azul marinho (#1B2A4A) para textos principais, dourado (#D4A843) para acentos/highlights
- Elementos futuristas: gradiente mesh animado no fundo, ícones com glow, progress rings, micro-animações
- Layout: grid assimétrico, cards de tamanhos diferentes, hero com saudação + stats em tempo real
- Módulos: cards grandes com gradiente interno, ícone flutuante com sombra colorida, hover com elevação 3D

## Elementos Chave
- Mesh gradient background (animated)
- Glassmorphism cards (backdrop-blur-xl, bg-white/60, border-white/20)
- 3D hover transforms (perspective, rotateX/Y)
- Animated gradient borders
- Floating icons with colored shadows
- Staggered entrance animations
- Large typography for numbers/stats
- Subtle particle effect or grid pattern
