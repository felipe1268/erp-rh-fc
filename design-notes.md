# Design Inspiration Notes - Hub de Módulos

## Key patterns from top SaaS platforms:

### 1. Greeting + Context (Monday, Notion, Linear)
- "Olá, Felipe!" personalized greeting
- Show current date/time
- Quick stats or recent activity

### 2. Card Grid with Visual Hierarchy (Vercel, Stripe, Linear)
- Large cards for primary modules (3-col grid)
- Each card has distinct color/gradient identity
- Hover reveals more info or subtle animation
- Clean white background, no clutter

### 3. Quick Actions Bar (Notion, Monday)
- Search bar prominent
- Quick shortcuts to recent items
- Notification bell

### 4. Visual Identity per Module
- Each module has its own color and icon
- Gradient or solid color header on card
- Number indicators (pending items, alerts)

### 5. Design Principles to Apply:
- White/light background - CLEAN
- Generous spacing
- Personalized greeting with user name
- Cards with gradient headers + white body
- Subtle shadows, no borders
- Quick stats on each module card (ex: "12 colaboradores", "3 EPIs vencendo")
- Smooth hover transitions
- Professional but warm
